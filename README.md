# Intrinsic Decomposition Google Colab Notebook
## このノートブックは、アップロードされた画像から固有反射成分（アルベド、スペックルフリー、拡散シェーディング）を自動的に分解し、結果を可視化・保存するためのツールです。

### 機能
#### 必要なライブラリのインストールとモデルのロード:   
Intrinsic Decompositionモデルに必要なPythonライブラリと事前学習済みモデルをセットアップします  
#### ローカルからの画像アップロード:  
あなたのローカルマシンから画像をColab環境にアップロードできます。現在は1枚の画像処理に最適化されています。  
#### Intrinsic分解の実行:   
アップロードされた画像に対してIntrinsic Decompositionモデルを適用し、アルベド、拡散シェーディング、残余成分を抽出します。  
#### スペキュラー無し画像の計算:   
抽出されたアルベドと拡散シェーディングからスペキュラ‐なし画像を生成します。  
#### 結果の表示と保存:   
オリジナル画像、スペキュラー無し画像、アルベド画像をノートブック内で可視化し、それらを自動的にローカルにダウンロードします。保存ファイル名は[元のファイル名]_[日付]_[タイプ].pngの形式になります。  
### 使用方法
#### ランタイムの設定:   
ランタイム > ランタイムのタイプを変更 > T4 GPU を選択して、GPUランタイムを設定します。  
#### セルを上から順に実行:  
ノートブックのすべてのコードセルを順番に実行してください。各セルの指示に従い、特に画像のアップロード時にはファイル選択ダイアログが表示されます。
### 注意事項
現在、img1 としてアップロードされた1枚の画像のみがIntrinsic分解パイプラインで処理されます。  
img2の取り扱い、または視差画像（disparity images）の使用に関する具体的な要件については、このノートブックでは対処されていません。追加の機能が必要な場合はお知らせください。
